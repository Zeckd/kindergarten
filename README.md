# Kindergarden

## Описание проекта
Kindergarden - это простое React приложение, созданное с использованием Vite. Это приложение предназначено для демонстрации основных возможностей React и Vite, а также для предоставления базовой структуры для дальнейшей разработки.

## Установка

1. Склонируйте репозиторий:
   ```
   git clone <URL_репозитория>
   ```

2. Перейдите в директорию проекта:
   ```
   cd kindergarden
   ```

3. Установите зависимости:
   ```
   npm install
   ```

## Запуск приложения

Для запуска всего приложения (Frontend + Backend + Database) используйте скрипт:

```bash
./start.sh
```

Этот скрипт:
1. Запустит PostgreSQL и Backend (Spring Boot) в Docker контейнерах.
2. Установит зависимости Frontend.
3. Запустит Frontend сервер.

Приложение будет доступно по адресу: `http://localhost:3000`

### Учетные данные по умолчанию
При первом запуске создается администратор:
- **Логин:** `admin`
- **Пароль:** `admin123`

Для остановки Backend части используйте:
```bash
./stop.sh
```

### Ручной запуск (для разработки)

**Backend:**
```bash
cd kindergarten
docker-compose up -d
```

**Frontend:**
```bash
npm install
npm run dev
```

## Структура проекта

- `kindergarten/` - Backend (Spring Boot) и конфигурация Docker.
- `public/` - содержит статические файлы, такие как изображения.
- `src/` - содержит исходный код Frontend приложения.
  - `api/` - конфигурация API клиента.
  - `assets/` - папка для хранения изображений и других ресурсов.
  - `components/` - папка для компонентов React.
  - `pages/` - страницы приложения.
  - `App.jsx` - основной компонент приложения.
  - `main.jsx` - точка входа приложения.
- `start.sh` - скрипт для быстрого запуска.
- `stop.sh` - скрипт для остановки backend.
- `vite.config.js` - конфигурация Vite (proxy настроен на порт 8080).
